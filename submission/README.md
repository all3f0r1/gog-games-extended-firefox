   - Privacy policy text

   - Step-by-step submission instructions
   - Follow this guide when submitting

## üöÄ Quick Start

1. Read **SUBMISSION_GUIDE.md** for detailed instructions
2. Go to https://addons.mozilla.org/developers/
3. Click "Submit a New Add-on"
4. Upload **gog-games-extended-1.3.1.zip**
5. Fill in information from **AMO_LISTING.md**
6. Add privacy policy from **PRIVACY_POLICY.md**
7. Upload screenshots (**before.png** and **after.png**)
8. Submit for review
