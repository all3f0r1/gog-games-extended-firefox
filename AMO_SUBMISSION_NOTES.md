# Firefox Add-ons (AMO) Submission Requirements

## Key Requirements

### File Format
- ZIP archive with extension `.zip`, `.xpi`, or `.crx`
- Maximum file size: 200 MB
- Must include all necessary files

### Required Information
1. **Name**: Extension name
2. **Add-on URL**: Unique URL on AMO (auto-generated from name, can be edited)
3. **Summary**: Short, descriptive summary
4. **Description**: Detailed features and functionality
5. **Categories**: Up to 2 categories for Firefox desktop and Android
6. **Support email/website**: Contact information for users
7. **License**: Choose appropriate license
8. **Privacy Policy**: Required if any data is transmitted from user's device

### Optional Flags
- **Experimental**: Mark if not ready for general use
- **Requires payment**: If add-on requires purchase/subscription

### Source Code
- Required if code is minified or obfuscated
- Must be submitted as separate package
- Should include build instructions

### Screenshots
- Recommended for better listing appeal
- Show extension functionality

### Review Process
- Automated validation first
- May require manual review
- Can take 24 hours or more
- Reviewer notes can be provided

## Our Extension Status

✅ No minification/obfuscation - source code submission NOT required
✅ No data collection - privacy policy optional (but recommended)
✅ Clean, readable code
✅ All files ready in repository
✅ Screenshots available (before/after)
